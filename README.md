# CSE 881 Project

**Given a graph dataset, we aim to build an algorithm for node classification.**

## Dataset
- Each node in the graph is associated with a feature vector and a label.
- No. of Nodes : 2480
- No. Of Edges : 10100
- No. of Classes : 7
- No. Features : 1390
- No. of Train Nodes : 496
- No. of Test None : 1984
